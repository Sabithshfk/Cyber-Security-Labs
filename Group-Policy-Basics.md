# Group Policy Basics

## What is Group Policy?

Group Policy is a feature in Microsoft Active Directory that allows administrators to centrally manage user and computer settings.

## Common Uses

- Password policies
- Desktop restrictions
- Software deployment
- Security settings
- Drive mappings
- Printer deployment

## Benefits

- Centralized management
- Improved security
- Consistent configurations across devices

## Example

An administrator can use Group Policy to require all users to:
- Use strong passwords
- Lock screens after inactivity
- Install approved software only

## Interview Question

### What is Group Policy?

Group Policy is a Microsoft feature used in Active Directory environments to centrally manage and enforce settings for users and computers across an organization.

### Why is Group Policy useful?

It helps administrators apply security, configuration, and software policies consistently across multiple devices from a central location.
