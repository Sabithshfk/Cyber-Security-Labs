# IT Support Interview Questions

## Tell me about yourself

Focus on:
- Current role
- Technical experience
- Career goals

---

## What is DNS?

DNS translates domain names into IP addresses.

Example:
google.com → IP address

---

## What is Active Directory?

A Microsoft directory service used to manage users, computers, and resources.

---

## What is Group Policy?

A feature used to centrally manage settings for users and computers.

---

## What is PXE Boot?

PXE allows a computer to boot from a network server when a local operating system is unavailable.

---

## User reports slow internet

1. Ask questions.
2. Determine scope.
3. Check connectivity.
4. Test DNS.
5. Escalate if required.

---

## Projector shows no display

1. Check power.
2. Verify cable.
3. Verify input source.
4. Check Win + P settings.
5. Test another cable/device.

---

## User cannot log in

1. Verify credentials.
2. Check account lockout.
3. Reset password if required.
4. Test login.
